# Blinkit-Grocery-Dashboard--Power-BI
# Retail Sales & Outlet Analytics Dashboard

## Project Description
An interactive Power BI dashboard built to visualize and analyze retail sales performance, outlet trends, item insights, and customer feedback metrics using real-world styled business data.

## Purpose
This project is designed to help stakeholders quickly understand key business metrics such as total sales, average sales, outlet performance, item categories, and customer ratings through intuitive visualizations — enabling better decision-making and operational insights.

## Features & Highlights
### Core Features
- Interactive Sales Overview showing total and average sales
- Outlet Trend Analysis to observe outlet performance over time
- Item & Category Insights to identify top-selling products
- Outlet Size & Type Comparison with layered visuals
- Customer Rating summaries for feedback insights
- Dynamic Filters to slice and dice data

### Highlights
- Clean, interactive dashboard visuals
- Drill-down analytics
- KPI cards for quick takeaways
- Designed for retail and business planning

## Tech Stack
- Power BI Desktop
- Power Query - (Cleab Data)
- Data Modeling
- DAX (Data Analysis Expressions)
- CSV / Excel
- Git & GitHub

## Data Source

The dashboard is built using structured **Excel files** exported from Blinkit’s internal business systems. These files contain the transactional sales records, outlet details, product information, and customer rating data required to generate insights in the dashboard.

- **Format:** Excel Workbook (`.xlsx`)  
- **Location in Repo (example):** `data/blinkit_sales_data.xlsx`
- **Imported via:** Power BI’s *Get Data → Excel*
- **Prepared using:** Power Query (for cleaning, transforming, and shaping data)

All data is processed using Power BI’s **Power Query** to ensure consistency and accuracy before being used in visuals and calculations.
